## Clonar repositorio

## Instalar dependencias

```bash
npm install
```

## Clonar .env.example

```bash
cp .env.example .env
```

## Levantar base de datos con docker

```bash
docker-compose up -d
```

## Levantar el proyecto

```bash
npm run start:dev
```
